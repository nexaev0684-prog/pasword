# 🔐 Генератор паролей (Password Generator)

Простая, но полезная программа на Python для создания надежных паролей. Использует символы из **ASCII-таблицы** (от 33 до 126), что включает в себя буквы (верхнего и нижнего регистра), цифры и специальные символы.

## ✨ Возможности

- Генерация пароля **любой длины** (которую вы укажете)
- Использование широкого набора символов (94 доступных символа)
- Простой и понятный интерфейс в командной строке

## 🚀 Как использовать

1. Убедитесь, что на вашем компьютере установлен **Python 3**.
2. Скачайте файл `import random.py` (название можно переименовать, например, в `password_generator.py`).
3. Откройте терминал или командную строку в папке с файлом.
4. Запустите программу:
   ```bash
   python "import random.py"
